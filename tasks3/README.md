## Overview

`tasks3` is a Python package for managing tasks and notes (a simple PKMS system). 
This iteration builds on `tasks2` by adding automated tests using pytest and organizing the code as a proper Python package.  

It includes:

- Increment function `inc(n)` for testing examples  
- `TaskManager` class for adding and completing tasks  
- `NotesManager` class for managing notes  
- Pytest-based automated tests for functions and classes  
- Can be run as a package using uv
