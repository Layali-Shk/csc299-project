# Tasks5 – Spec-Kit Task Manager

This is Task 5 for CSC 299. It is a **task manager** built using **GitHub Spec-Kit** and follows a **spec-driven development workflow**. The system allows users to add, list, complete, and delete tasks, all stored in memory.

## Features
- Add tasks with unique IDs
- List all tasks
- Complete tasks to mark them as done
- Delete tasks
- Fully tested with pytest

## Project Structure
- `github/agents/` – Spec-Kit agent files
- `specify/memory/constitution.md` – Project constitution
- `specify/scripts/bash/` – Spec-Kit utility scripts
- `specify/templates/` – Spec-Kit templates
- `scripts/run.sh` – Run the task manager
- `specs/` – Specifications
- `src/` – Source code
- `tests/` – Unit tests
- `.gitignore`, `pyproject.toml`, `README.md`
